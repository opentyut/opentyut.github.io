# 外部资源聚合

这里不是越多越好的收藏夹，而是给刚开始自学的同学准备的入口页。先选一个方向，用一两个高质量资源坚持学完，比同时打开十几个标签页更有效。

::: tip 怎么用这页
如果你还没有明确方向，先看「在线学习平台」和「推荐视频课程」；如果你已经在准备面试或竞赛，优先看「刷题平台」；如果你想参与项目，先把 Git、GitHub 和编辑器配好。
:::

## 在线学习平台

### CS50

- 链接：[cs50.harvard.edu](https://cs50.harvard.edu/)
- 适合：想系统了解计算机科学的新生
- 建议：不要只看视频，尽量跟着完成作业；这门课适合当作计算机世界的第一张地图。

### 中国大学 MOOC

- 链接：[icourse163.org](https://www.icourse163.org/)
- 适合：想补国内高校课程、专业基础课的同学
- 建议：适合配合校内课程使用，尤其是高数、线代、数据结构等基础课。

### 慕课网

- 链接：[imooc.com](https://www.imooc.com/)
- 适合：想看前端、后端实战项目的同学
- 建议：挑一个项目完整做完，比零散看很多课程更有收获。

### Bilibili

- 链接：[bilibili.com](https://www.bilibili.com/)
- 适合：快速入门、找中文讲解、查某个具体概念
- 建议：注意筛选发布时间和评论反馈，避免跟着过时教程配置环境。

## 刷题平台

### LeetCode

- 链接：[leetcode.cn](https://leetcode.cn/)
- 适合：准备校招、实习面试的同学
- 建议：先刷数组、链表、栈、队列、哈希表、二叉树等高频基础题，不要一开始追求难题数量。

### 洛谷

- 链接：[luogu.com.cn](https://www.luogu.com.cn/)
- 适合：算法竞赛入门和提高
- 建议：适合按题单推进，重点训练算法思维和代码严谨性。

### NowCoder

- 链接：[nowcoder.com](https://www.nowcoder.com/)
- 适合：准备校招笔试、面经复盘
- 建议：更适合临近求职时使用，平时不要把刷题当成学习的全部。

## 开发工具

### GitHub

- 链接：[github.com](https://github.com/)
- 用途：代码托管、协作、开源贡献
- 建议：大一就应该学会 Git 和 GitHub，至少能完成 clone、commit、push 和 Pull Request。

### GitHub Student Pack

- 链接：[education.github.com](https://education.github.com/pack)
- 用途：学生免费工具包
- 建议：用学校邮箱认证，能领取 GitHub Pro、JetBrains、云服务额度等福利。

### VS Code

- 链接：[code.visualstudio.com](https://code.visualstudio.com/)
- 用途：主力代码编辑器
- 建议：先保持插件克制，等真正遇到需求再安装扩展。

### JetBrains 教育版

- 链接：[jetbrains.com/student](https://www.jetbrains.com/community/education/)
- 用途：IDE 全家桶
- 建议：适合 Java、后端、Python 等开发场景，学生认证后可以免费使用。

## 技术博客与社区

### 掘金

- 链接：[juejin.cn](https://juejin.cn/)
- 适合：看中文技术文章、工程实践经验

### SegmentFault

- 链接：[segmentfault.com](https://segmentfault.com/)
- 适合：查中文问答和技术文章

### V2EX

- 链接：[v2ex.com](https://www.v2ex.com/)
- 适合：观察开发者社区讨论和行业日常

### Stack Overflow

- 链接：[stackoverflow.com](https://stackoverflow.com/)
- 适合：查英文技术问题和报错
- 建议：学会用英文关键词搜索，是自学效率的分水岭。

## 推荐项目

大一学会 Git 之后，可以看看这些适合新手的开源项目：

- [freeCodeCamp](https://github.com/freeCodeCamp/freeCodeCamp)：免费学编程，内容体系完整。
- [build-your-own-x](https://github.com/codecrafters-io/build-your-own-x)：从零造轮子合集，适合想理解底层原理的同学。
- [public-apis](https://github.com/public-apis/public-apis)：免费 API 列表，适合练手做小项目。

::: tip GitHub Student Pack
用学校邮箱注册 GitHub 可以免费领 GitHub Pro、JetBrains 全家桶、域名、云服务额度等几十种福利。大一就可以领。
:::
