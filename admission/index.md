# 升学指南

升学是许多同学关注的方向。这里整理了保研、考研和留学三条路径的经验，希望能帮助你做出适合自己的选择。

::: tip 提示
每一条路都没有绝对的好坏，关键是结合自己的兴趣、能力和目标来做决定。
:::

## 先选你要了解的路径

### 保研

- 适合：成绩、科研、竞赛或综合表现有一定积累，并希望继续读研的同学。
- 先看：[保研概览](/admission/exemption_of_examinations)

### 考研

- 适合：目标院校或目标专业比较明确，愿意投入长期复习的同学。
- 先看：[考研概览](/admission/postgraduate_qualifying_examination)

### 留学

- 适合：愿意提前准备语言、绩点、科研实习和申请材料，并能承担相应成本的同学。
- 先看：[留学](/admission/study-abroad)

## 先想清楚为什么升学

升学不是唯一正确答案，也不是逃避就业焦虑的保险箱。在投入大量时间之前，建议先想清楚：你想获得什么能力、去什么环境、解决什么问题。

如果还没有答案，可以先读 [你想要做什么](/survival/li-zhi/what-to-do)。
