## 适合谁

后端开发适合喜欢系统设计、数据建模、接口设计和性能问题的同学。它不只是在服务器上写代码，还包括让服务稳定、可维护、可扩展地运行。

## 后续会补什么

- 编程语言：Java、Go、Python 或 Node.js 的选择建议。
- 计算机基础：操作系统、计算机网络、数据库和数据结构。
- Web 后端：HTTP、认证鉴权、日志、异常处理、接口设计。
- 工程实践：测试、部署、监控、性能优化和团队协作。
